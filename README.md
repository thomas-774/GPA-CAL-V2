🎓 GPA Calculator Website

A simple and user-friendly GPA Calculator that helps students calculate their GPA easily using two different grading systems.

🚀 Features
Calculate GPA using letter grades (A, B, C, etc.)
Calculate GPA using numeric scores (e.g., 95, 88, etc.)
Supports GPA scale out of 4.0
Clean and responsive UI
Easy to use and fast performance
🛠️ Built With
HTML
CSS
JavaScript

⚙️ How to Use

Input your grades
Click on Calculate GPA
Get your result instantly
